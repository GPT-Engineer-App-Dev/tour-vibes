# tour-vibes

Create a landing page for a band's touring website with the following:

1. Header with a placeholder for the band's name and logo
2. Hero section with a large placeholder image
3. Basic layout and styling using the Chakra UI components

Use a clean, modern design with a dark color scheme suitable for a music website.
